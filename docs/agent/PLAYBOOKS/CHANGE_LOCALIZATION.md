# PLAYBOOK: CHANGE LOCALIZATION

## Когда использовать
- Добавление/изменение пользовательских строк.

## Шаги
1. Найти релевантный `Localizable.xcstrings` в модуле.
2. Добавить/обновить ключи в каталоге строк.
3. Использовать в UI только `String(localized:)`.
4. Проверить, что нет нового hardcoded текста.
5. При необходимости обновить localization-тесты.

## Чеклист PR
- Нет строк литералами в UI.
- Ключи консистентны между модулями.
